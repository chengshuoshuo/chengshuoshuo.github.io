数据间的一些转化方法

String()、Number()、Boolean()可以将数据强行转化
Number()在将字符串转为数字类型时会将引号去掉，如果还有字母类的字符，值为NaN；将布尔值转化为数字类型时，true是1，false是0；
Boolean()将字符串转化为布尔值时，字符串为空就是false，字符串有值就是true；underfind类型、null类型、NaN类型的数据由由Boolean()转化来就是false

数据间的运算

string+number=string
字符串和数字间的加法是将数字转化为字符串来进行拼接

string+boolean=string
将布尔类型转化为字符串来与之拼接

number+boolean=number
数字类型和布尔类型的加法是将布尔类型转化成数字类型进行运算

当运算符两侧有一侧是数字类型，另一侧是underfind或者NaN时，结果为NaN

数字类型数据的一些操作方法

  • parseInt()小数点取整数

  • parseFloat()从第一个字符开始解析直到最后一个，有两种情况会停下解析

    • 第一、是遇到无效的浮点、如空格、小数点

    • 第二、是遇到非数字的字段

      • 空串就返回NaN

      • 字母e、解析e前面的数字乘以10的n次方，n是e后面的数字，最后返回这个值

  • Math.random()，随机值，大于等于0，小于1

  • Math.floor()向下取整数，效果同parseInt()

  • Math.abs()取数字的绝对值

  • Math.min(x,y,z)获取其中最小的数

  • Math.max()获取其中最大的数

  • Math.round(x)四舍五入

  • Math.PI获取圆周率

数组(Array)的一些操作方法
  • push(string) 在数组的尾部添加元素并返回数组长度
  • pop() 将数组的尾部元素删除并返回那个被删除的元素
  • unshift(string)在数组的头部添加元素并返回数组长度
  • shift() 将数组的头部元素删除并返回那个被删除的元素
  • slice(number1,number2)获取数组内位置从number1到位置number2(不包含number2)的元素
  • splice(number1,number2)获取数组中位置从number1开始的number2个元素
  • join(string)将数组内的元素转为字符串形式拼接来，string是拼接时元素间的添加元素
  • concat(Array)将一个数组和并在另一个数组中并可以赋予给新的变量
  • sort()数组内的元素按顺序排序
  • reverse()数组内元素反序排列
  • lenght获取数组长度
