常见的事件

常见的事件有以下几种

  • 鼠标单击事件(onclick())
  • 鼠标经过事件(onmouseover)
  • 鼠标移开事件(onmouseout)
  • 光标聚集事件(onfocus)
  • 失去焦点事件(onblur)
  • 内容选中事件(onselect)
  • 文本框内容改变事件(onchange)
  • 加载事件(onload)
  • 卸载事件(onunload)
