javascript-入门之运算符

运算符的定义

运算符可以大致分类

  • 算数运算符
    • 加法运算 (+)
    • 减法运算 (-)
    • 乘法运算 (*)
    • 除法运算 (/)
    • 求余运算 (%)
  • 比较运算符
    • 大于(>)
    • 大于或等于(>=)
    • 小于(<)
    • 小于或等于 (<=)
    • 值相等 (==)
    • 类型和值都相等(===)
  • 逻辑运算符
    • 逻辑非(！)
    • 逻辑与(&&)
    • 逻辑或(||)
  • 赋值运算符
    • 赋予变量值(=)
    • 对变量和表达式执行加法运算，结果赋给变量(+=)
    • 对变量和表达式执行减法运算，结果赋给变量(-=)
    • 对变量和表达式执行乘法运算，结果赋给变量(*=)
    • 对变量和表达式执行除法运算，结果赋给变量(/=)
  • 位运算符
    • 对两个表达式执行按位“与”运算(&)按位与运算
    • 对两个表达式执行按位“或”运算(|)按位或运算
    • 按位右移运算符(>>)
    • 按位右移运算符(<<)

运算符的优先级

小括号>算数>比较>逻辑>赋值
