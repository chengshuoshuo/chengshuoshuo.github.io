html简介

HTML是什么
  • html hyer text markup language 超文本标记语言
  • xml 可扩展的标记语言
  • html是xml的扩展语言 用于设计语言 不算是编程语言
  • html5是html的第五个版本 他相比之前扩展了一些新功能
  • 狭义上讲html5是html的第五个版本 广义上将html5指的是新一代web开发技术的综和 包括html5 css3 es6
  • 纯文本文件 文件中所有数据都是文本数据 例如txt
  • 开发工具 vscode轻量级编辑器 便于安装 丰富的可扩展插件 谷歌 火狐 等
  • 一个网页本质就是html文件 包含一段html文本 本质就是一个纯文本文件 文本编辑器可以直接打开 看到源代码
